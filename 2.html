SET UP
1.Club 2.Square Stance 3.Grip 4.Body 정렬 제자리(① Take-awy,② Top, Transition), 회전(③ Impact, Finish ) 프리샷루틴
1 볼 뒤에서 목표에서 볼까지 일직선을 긋고 가상목표(30cm앞)를 설정후, 티를 꽂고 공을 올려놓는다. 볼 옆으로 가서 연습스윙 (모아스윙, 연속스윙)을 3회하면서, 스윙의 리듬과 템포를 파악한다. Ball 뒤로 가서 가상의 Target ( 30cm앞 )을 다시 확인후 SET-UP한다. - 가상목표에 확신이 없으면 다시 확인한다.
2 크럽정렬 - 클럽을 오른손 한손으로 가상의 Target ( 30cm앞 )과 수직정렬후 왼손을 붙여서 그립을 잡는다. Square Stance - 어깨넓이, Ball-가운데/ <왼발앞,뒤꿈치를-안,밖,그대로>,<오른발-안,밖,그대로>하여 <넓게,좁게,그대로> 선다. Body 정렬 - 등을 곧게하고, 양팔은 몸통에, 체중은 오른발 앞꿈치, 턱을 당긴다. Strong grip후 Waggle하면서 시선은 Ball의 오른쪽 밑면을 주시한다.
풀 스윙 45도 들어, 올리고 - Take-awy, Back swing Top ( 왼어깨가 턱을 감싸듯이 왼팔들어 오른귀로 올리고, 왼다리 내전근 버티고 ) 15도 밀고, 내려 - Down swing ( 코어를 밀고, 오른귀 내리며, 클럽을 "딱"내려치고 < 하이파이브> ) 높이 올리고 - Finish ( 양손을 왼쪽귀로 높이 올려, 무릎 붙히며 하이피니시 )
