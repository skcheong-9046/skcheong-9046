GOLF란 ?
- 스윙이 아니고, 집중이며, 몰입이고, 습관이며, 전략이다.
SWING이란 ?
- 제자리회전이며 하체로 휘두르며, 왼쪽에서 소리나게, 원을 그리는 것으로, 빨라지면서 지나치는 것이다 착한스윙이란 ? - 내몸에 어울리는 스윙으로, 스코어를 내기위한 수단으로서 스윙이다. 임펙트란 ? - 히팅이 아니라 다운스윙의 한 과정이다.( 과도한 힘을 자제하라. ) Putting ? : 등으로- 밟고 양팔을 몸에 붙이고, 왼등으로 백스윙, 오른등으로 다운스윙, 오른발바닦에 체중을 남긴다. 기준거리 퍼팅을 ( 10cm, 3m, 5발 / 60cm, 18m, 30발 ) 확보하라. 연습이란 ? - 샷의 경향을 파악하고, 집중과 몰입능력을 향상하는 것이다.

- 1차 : 오른 무릎 ( 테이크어웨이 ) - 오른발 발바닦 안쪽 - 2차 : 오른 발바닦 ( 백스윙탑, 전환 ) -오른발 발바닦(뒤), 왼발 앞꿈치 - 3차 : 왼 무릎 ( 다운스윙, 임펙트 ) - 왼발 발바닦(앞), 오른발 앞꿈치 - 4차 : 오른 엉덩이 ( 피니시 ) - 왼발 발바닦(뒤) 스윙의 궤도 IN에서 OUT ( 6시30분에서 12시30분방향 ) 으로 피니시 임펙트후에 볼을 맞고난 후 1m정도 헤드를 더 빨리 접어 올린다. 어프로치 시 볼은 오른발앞에, 손은 볼보다 앞쪽에, 손목아닌 어깨로 회전하라, 하체를 고정하라. 몸안에서 스윙하라, 삼각형을 유지하라, 왼팔을 펴라 띄워야 할때 공의위치-스텐스 중앙, 손의위치-핸드퍼스트, 스텐스폭-조금넓게 굴려야 할때 공의위치-오른발쪽에, 손의위치-Y자형어드레스, 스텐스폭-조금좁게 높이 띄워야 할때 공의위치-왼발쪽, 스텐스-오픈, 슬라이스교정 - 스텐스정렬후 오른발 뒤로, 훅교정 - 스텐스정렬후 왼발뒤로, 뒷땅, 탑핑교정 - 뒷땅 ( 백스윙을 조금짧게 ), 탑핑 ( 백스윙을 조금크게 ) 중력을 이용한 시계추 삼각형 스윙 머리를고정방법 - 스윙시 왼쪽머리를 타이거우즈가 손가락을 대고 있다고 생각하라. 다운스윙의 시작부터 팔로스루까지 시선은 공에 고정하고, 30CM앞에 있는 볼까지 쳐낸다는 느낌으로 스윙한다. 방향성 향상방법 ? -30cm의 의미를 기억하라. 왼쪽에 장애물이 있거나, 왼쪽 도그래그홀시 티박스 왼쪽에서 서서 페어웨이 중앙을 목표로한다. 오른쪽 장애물이 있거나 오른쪽 도그래그홀시 티박스 오른쪽에서서 페어웨이 중앙을 목표로 한다. 롱아이언, 오르막 시 티를 높게(풀높이)하고, 숏아이언, 내리막 시 티는 낮게(땅높이)한다. 맞 바람엔 스윙의 크기를 80%로하며 조금 더 부드럽게 스윙 한다. 헤드업을 방지하려면 눈을 크게뜨고 볼을 주시한다. 그립을 잡기전에 왼손가락 약지와 새끼를 뒤로 젖히고 그립을 잡는다. 잘하는 것을 중심으로 안되는 샷을 보완하라. 타킷은 다음샷을 하기에 좋은곳으로 하고, 최소한의 위험을 피하라. 오르막지형에서는 티를 높게하고 내리막지형에서는 티를 낮게하라. 배꼽아래는 능동적으로, 배꼽 위는 수동적으로, 숏게임 정복, 때리지 말고 휘둘러라. 퍼팅 실패해도 괜찮을 지점까지 보내라. - 홀을 보면서 스트로크 연습 굴러가는 공을 상상하라. - 페이스를 스퀘어로 유지하라, 페이스 중앙에 맞혀라. - STANCE : 원칙은 스궤어, 시선이 바깥으로 가면 OPEN, 안쪽으로 가면 CLOSE로 한다 : UPPER BLOW 타법, 등이 곧게 선자세 ( 원호타격 ), 등이 구부러진 자세 ( 일직선 타격 ) : 볼은 중앙정렬,클럽은 중앙 오른쪽정렬 : W 정렬 ( 양발 끝, 볼, 목표 방향 앞뒤 ) - 스트로크 시계추 운동이며, 양팔꿈치를 붙이고, 엄지손톱을 퍼터와 같이 가상의 목표로 보내며, 퍼터그립의 끝을 몸의 중심에 두고, 퍼터와 볼의 맞주침을 눈으로 본다. - putter와 elbow가 하나되어 스트로크 한후 3초동안 머리를 고정하고 피니시 자세를 유지한다. - 헤드와 양손등이 같이움직인다는 느낌으로 밀어치며, 이때 왼머리와 오른어깨는 잡아둔다. - 오르막퍼팅 : 홀뒤를 맞히겠다는 생각으로 강한 스트로크을 하며, 피니시를 길게 한다. - 내리막퍼팅 : 운동력이 다소멸 한 체 마지막 회전에서 굴러 떨어지게 하며 피니시를 짧게 한다. - 훅라이퍼팅 : 손목을 조금 곧게 세우기 위하여, 상체를 약간 더 세우고 퍼팅 - 슬라이스라이퍼팅 : 손목을 조금 눌러주기 위하여, 상체를 더 숙이고 퍼팅 비오는날 1. 티샷은 3번우드로 2. 아이언으로 쓰리온을 ( 우드일경우 - ball은 오른쪽 2개 ) 3. 어프로치시 볼의 위치는 오른쪽으로 2개 4. 퍼팅은 홀보고 강하게 연습방법 시작 10분이 나의 현재 실력이다. 프리샷 루틴을 철저히 연습하라 클럽을 양손에 들고, 체중이동-회전-체중이동-회전을 한다. ( 스트레칭 ) 양발을 모은후 백스윙시 왼발을 벌리면서 스윙하기. ( 전환 ) 한손(왼손, 오른손)으로 스윙하고, 한손(왼손, 오른손)으로 퍼팅하자. 하체(오른팔은 배앞으로)로 휘둘러서 소리내기 ( 1단계 - 10단계로 ) 왼발 발바닦으로 체중이동 하기 ( 한발로 서기 ) 눈감고 빈스윙 하루 500번 하자 하체강화 - 등산, 계단 오르고 내리기

감사하는 마음을 같고, 스윙의 완성보다는 스코어 향상에 집중하며, 나만의 연습법을 만들자. 잘되는 것을 중심으로 안되는 샷을 보완하며. 연습장밖에서 연습하고, 루틴을 연습하자.
