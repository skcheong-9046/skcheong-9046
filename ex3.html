-30cm의 의미를 기억하라. 왼쪽에 장애물이 있거나, 왼쪽 도그래그홀시 티박스 왼쪽에서 서서 페어웨이 중앙을 목표로한다. 오른쪽 장애물이 있거나 오른쪽 도그래그홀시 티박스 오른쪽에서서 페어웨이 중앙을 목표로 한다. 롱아이언, 오르막 시 티를 높게(풀높이)하고, 숏아이언, 내리막 시 티는 낮게(땅높이)한다. 맞 바람엔 스윙의 크기를 80%로하며 조금 더 부드럽게 스윙 한다. 헤드업을 방지하려면 눈을 크게뜨고 볼을 주시한다. 그립을 잡기전에 왼손가락 약지와 새끼를 뒤로 젖히고 그립을 잡는다. 잘하는 것을 중심으로 안되는 샷을 보완하라. 타킷은 다음샷을 하기에 좋은곳으로 하고, 최소한의 위험을 피하라. 오르막지형에서는 티를 높게하고 내리막지형에서는 티를 낮게하라. 배꼽아래는 능동적으로, 배꼽 위는 수동적으로, 숏게임 정복, 때리지 말고 휘둘러라. 퍼팅 실패해도 괜찮을 지점까지 보내라. - 홀을 보면서 스트로크 연습 굴러가는 공을 상상하라. - 페이스를 스퀘어로 유지하라, 페이스 중앙에 맞혀라. - STANCE : 원칙은 스궤어, 시선이 바깥으로 가면 OPEN, 안쪽으로 가면 CLOSE로 한다 : UPPER BLOW 타법, 등이 곧게 선자세 ( 원호타격 ), 등이 구부러진 자세 ( 일직선 타격 ) : 볼은 중앙정렬,클럽은 중앙 오른쪽정렬 : W 정렬 ( 양발 끝, 볼, 목표 방향 앞뒤 ) - 스트로크 시계추 운동이며, 양팔꿈치를 붙이고, 엄지손톱을 퍼터와 같이 가상의 목표로 보내며, 퍼터그립의 끝을 몸의 중심에 두고, 퍼터와 볼의 맞주침을 눈으로 본다. - putter와 elbow가 하나되어 스트로크 한후 3초동안 머리를 고정하고 피니시 자세를 유지한다. - 헤드와 양손등이 같이움직인다는 느낌으로 밀어치며, 이때 왼머리와 오른어깨는 잡아둔다. - 오르막퍼팅 : 홀뒤를 맞히겠다는 생각으로 강한 스트로크을 하며, 피니시를 길게 한다. - 내리막퍼팅 : 운동력이 다소멸 한 체 마지막 회전에서 굴러 떨어지게 하며 피니시를 짧게 한다. - 훅라이퍼팅 : 손목을 조금 곧게 세우기 위하여, 상체를 약간 더 세우고 퍼팅 - 슬라이스라이퍼팅 : 손목을 조금 눌러주기 위하여, 상체를 더 숙이고 퍼팅 비오는날 1. 티샷은 3번우드로 2. 아이언으로 쓰리온을 ( 우드일경우 - ball은 오른쪽 2개 ) 3. 어프로치시 볼의 위치는 오른쪽으로 2개 4. 퍼팅은 홀보고 강하게 연습방법 시작 10분이 나의 현재 실력이다. 프리샷 루틴을 철저히 연습하라 클럽을 양손에 들고, 체중이동-회전-체중이동-회전을 한다.
